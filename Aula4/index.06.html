Exemplo 1

HTML (HyperText Markup Language) é a linguagem de marcação padrão para estruturar páginas web, definindo elementos como parágrafos, links, imagens e títulos. Não é uma linguagem de programação, mas sim um código lido por navegadores (Chrome, Firefox, Safari) para exibir o conteúdo visual, servindo como o esqueleto de sites.
